# WP Rocket | Disable Google Font Optimization

Disable Google Font Optimization in WP Rocket. To re-enable the option, just disable the plugin.

Documentation:
* [Optimize Google Fonts](https://docs.wp-rocket.me/article/1312-optimize-google-fonts)

To be used with:
* Any setup where Google Font Optimization is causing conflicts and needs to be disabled. Since 3.7 Google Font Optimization is enabled by default and there is no option in WP Rocket UI to disable this feature. If Google Font Optimization was disabled prior to 3.7 or after using this helper plugin, it can be re-enabled disabling this helper. 

Last tested with:
* WP Rocket {3.20.x}
* WordPress {6.9.x}
